id 选择器

id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式。

id 选择器以"#"来定义。

下面的两个 id 选择器，第一个可以定义元素的颜色为红色，第二个定义元素的颜色为绿色：

  1. #red {color: red;}
  2. #green {color: green;}

下面是 HTML 代码中，id 属性为 red 的 p 元素显示为红色，而 id 属性为 green 的 p 元素显示为绿色。

  1. <p id="red">这个段落是红色。</p>
  2. <p id="green">这个段落是绿色。</p>

注意：id 属性只能在每个 HTML 文档中出现一次。(详情参阅：XHTML:网站重构。)

id 选择器和派生选择器

在现代布局中，id 选择器常常用于建立派生选择器。

  1. #sidebar p {
  2.     font-style: italic;
  3.     text-align: right;
  4.     margin-top: 0.5em;
  5.     }
  7. #sidebar h2 {
  8.     font-size: 1em;
  9.     font-weight: normal;
  10.     font-style: italic;
  11.     margin: 0px;
  12.     line-height: 1.5em;
  13.     text-align: right;
  14.     }

在这里，与页面中的其他 p 元素明显不同的是，siderbar 内的 p 元素得到了特殊处理，同时，与页面中其他所有 h2 元素明显不同的是，siderbar 中的 h2 元素也得到了不同的特殊处理。

单独的选择器

id 选择器即使不被用来创建派生选择器，它也可以独立发挥作用：

  1. #sidebar {
  2.     border: 1px dotted #000;
  3.     padding: 10px;
  4.     }

根据这条规则，id 为 sidebar 的元素将拥有一个像素的黑点状边框，同时其周围会有10个像素宽的内边距(padding,内部空白)。老版本的 Windows/IE 浏览器可能会忽略这条规则，除非我们特别地定义这个选择器所属的元素：

  1. div#sibedar {
  2.     border: 1px dotted #000;
  3.     padding: 10px;
  4.     }
