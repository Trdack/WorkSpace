HTML_属性

属性为HTML元素提供附加信息。

HTML属性

HTML元素可设置属性。

属性可以在元素中添加附加信息。

HTML标签可以拥有属性。属性提供了有关HTML元素的更多的信息。

属性总是以名称/值对的形式出现，比如：name="value"。

属性总是在HTML元素的开始标签中规定。

<标签 属性="值">内容</标签>

属性实例

HTML链接由<a>标签定义。链接的地址在href属性中指定：

<a href="http://www.baidu.com ">这是百度的链接</a>

更多HTML属性实例

属性实例1：

<h1>定义标题的开始。

<h1 algin="center">拥有关于对其方式的附加信息。

<h1 algin="center">标题H1</h1>

注释：algin="center"表示居中对齐。

属性实例2：

<body>定义HTML文档的主体。

<body bgcolor="yellow">拥有关于背景颜色的附加信息。

<body bgcolor="yellow">

<h1 algin="center">背景属性</h1>

<p>bgcolor是背景属性，yellow是属性值，表示黄色。</p>

</body>

注释：bgcolor="yellow"表示背景颜色为黄色。

属性实例3：

<table>定义HTML表格。(您将在稍后的章节学习到更多有关HTML表格的内容)

<table border="1px">拥有关于表格边框的附加信息。

注释：border定义元素的边框属性，1px表示线宽，px(单位)表示像素。

HTML提示：使用小写属性

属性和属性值对大小写不敏感。

不过，万维网联盟在其HTML4推荐标准中推荐小写属性/属性值。

而新版的(X)HTML要求使用小写属性。

始终为属性值加引号

属性值应该始终被包括在引号内。

双引号是最常用的，不过使用单引号也没有问题。

提示：在某些个别的情况下属性值本身就含有双引号，那么您必须使用单引号。

例如：

name='Bill"HelloWorld"Gates'
