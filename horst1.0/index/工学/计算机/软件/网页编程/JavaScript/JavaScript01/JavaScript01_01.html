JavaScript 简介

=>> 网页编程 -> JavaScript -> JavaScript 简介

JavaScript 是世界上最流行的语言。

这么语言可用于 HTML 和 Web ，更可广泛用于服务器、PC、笔记本电脑、平板电脑和智能手机等设备。

JavaScript 是脚本语言

JavaScript 是一种轻量级的编程语言。

JavaScript 是可插入 HTML 页面的编程代码。

JavaScript 插入 HTML 页面后，可由所有现代浏览器执行。

JavaScript 很容易学习。

  • HTML 文档描述网页
  • HTML 文档包含 HTML 标签和纯文本
  • HTML 文档也被称为网页
